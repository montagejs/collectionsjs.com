collections.js

This package provides JavaScript implementations of common collections, with many useful features.

findLastIndex(callback, thisp?)

(not implemented) Finds the last index within a collection that passes a test, searching from the right.

At time of writing, no collection implements this method, but it would be be a logical addition triangulated from findIndex (an ECMA6 Proposal) and lastIndexOf.
